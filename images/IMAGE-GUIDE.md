# Image Asset Guide

Drop your images into the folders below. The filenames listed are what the HTML expects after the replacement is made.

## images/logo/
| Filename | Dimensions | Used In | Description |
|---|---|---|---|
| `logo.svg` or `logo.png` | ~180x28 (or any width, height auto-scales) | index.html & pdp.html — Navigation bar | Main Wrightsock logo. SVG preferred for crisp scaling. Also appears in footer if needed. |

## Tips
- Use `.jpg` for photos, `.png` for anything needing transparency (like the sock diagram)
- Optimize images before adding (aim for <200KB per image, <500KB for hero)
- All dimensions above are the minimum recommended size — larger is fine, the CSS handles scaling
